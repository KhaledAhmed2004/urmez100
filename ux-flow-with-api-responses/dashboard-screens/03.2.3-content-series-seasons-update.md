# Screen 3.2.3: Edit Season Content (Update)

> **Section**: Dashboard APIs (Admin-Facing)
> **Base URL**: `{{baseUrl}}` = `http://localhost:5000/api/v1`
> **Response format**: See [Standard Response Envelope](../README.md#standard-response-envelope)
> **Related screens**: [Get Season Content](./03.2.2-content-series-seasons-list.md), [Create Series & Seasons](./03.2.1-content-series-create.md)

## Overview

This screen allows admins to **Edit Season Content**, including updating its metadata (Title, Season Number) and replacing its media files (Poster, Trailer/Teaser).

---

### 1.1 Edit Season Content
```http
PATCH /admin/series/seasons/:seasonId
Authorization: Bearer {{accessToken}} (SUPER_ADMIN)
Content-Type: multipart/form-data
```

**Business Logic:**
- **Update**: Allows partial updates to season details.
- **Media**: Admins can upload a new `posterFile` or `trailerFile` to update the season's visuals and teaser.
- **Normalization**: Automatically handles file storage and URL generation.

**Request Fields (Form-Data):**
- `title`: String
- `seasonNumber`: Number
- `posterFile`: File
- `trailerFile`: File (Season Teaser)

**Response:**
```json
{
  "success": true,
  "statusCode": 200,
  "message": "Season content updated successfully",
  "data": {
    "_id": "664a1b2c3d4e5f6a7b8c9d55",
    "title": "Season 1: Updated Title",
    "poster": "https://uremz-video-stream.b-cdn.net/posters/new-poster.jpg",
    "trailer": "https://uremz-video-stream.b-cdn.net/trailers/new-teaser.mp4"
  }
}
```

---

### 1.2 Delete Season
```http
DELETE /admin/series/seasons/:seasonId
Authorization: Bearer {{accessToken}} (SUPER_ADMIN)
```

**Business Logic:**
- **Destructive Operation**: Permanently removes the season.
- **Cascade Check**: Usually, deleting a season should also handle the episodes under it (either by deleting them or preventing deletion if episodes exist).
- **Sync**: Recalculates `seasonsCount` on the parent Series document.

**Response:**
```json
{
  "success": true,
  "statusCode": 200,
  "message": "Season deleted successfully"
}
```
