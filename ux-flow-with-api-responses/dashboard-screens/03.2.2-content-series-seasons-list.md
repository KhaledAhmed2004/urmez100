## Overview

This API is used to **Get Seasons List**, specifically retrieving all seasons associated with a specific series. This is required for the series management dashboard to show the season hierarchy.

---

### 1.1 Get Seasons List
```http
GET /admin/series/:seriesId/seasons
Authorization: Bearer {{accessToken}} (SUPER_ADMIN)
```

**Business Logic:**
- **Source**: Fetches all seasons linked to the provided `seriesId`.
- **Sorting**: Ordered by `seasonNumber` (ASC).
- **Data**: Returns season metadata (Title, Number, Poster, Trailer).

**Response:**
```json
{
  "success": true,
  "statusCode": 200,
  "message": "Seasons list retrieved successfully",
  "data": [
    {
      "_id": "664a1b2c3d4e5f6a7b8c9d55",
      "title": "Season 1: The Beginning",
      "seasonNumber": 1,
      "poster": "https://uremz-video-stream.b-cdn.net/posters/witcher-s1.jpg",
      "trailer": "https://uremz-video-stream.b-cdn.net/trailers/witcher-s1.mp4",
      "seriesId": "664a1b2c3d4e5f6a7b8c9d1a",
      "episodeCount": 8
    }
  ]
}
```
