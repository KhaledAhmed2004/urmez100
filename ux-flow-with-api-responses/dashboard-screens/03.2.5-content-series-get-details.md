## Overview

This API retrieves the high-level details of a specific series, including its metadata, seasons list, and overall episode counts. This is typically used when an admin clicks "View" or "Edit" on a series.

---

### 1.1 Get Series Details
```http
GET /admin/series/:seriesId/details
Authorization: Bearer {{accessToken}} (SUPER_ADMIN)
```

**Business Logic:**
- **Source**: Fetches the `Content` document for the given `seriesId`.
- **Aggregation**: Includes the list of associated seasons and counts for episodes.
- **Data**: Returns title, description, poster, trailer, genres, cast, and season hierarchy.
