## Overview

This API is used to **Get Season Content**, specifically retrieving all episodes associated with a specific season. This is used in the dashboard to manage episodes under a season.

---

### 1.1 Get Season Episodes
```http
GET /admin/series/:seriesId/episodes?seasonId=664a1b2c3d4e5f6a7b8c9d55
Authorization: Bearer {{accessToken}} (SUPER_ADMIN)
```

**Business Logic:**
- **Source**: Fetches episodes for a specific series, filtered by `seasonId`.
- **Pagination**: Supports `page` and `limit`.
- **Sorting**: Ordered by `episodeNumber`.

**Query Parameters:**
- `seasonId`: (Required) The ID of the season.
- `page`: Page number.
- `limit`: Items per page.

**Response:**
```json
{
  "success": true,
  "statusCode": 200,
  "message": "Season episodes retrieved successfully",
  "pagination": { "page": 1, "limit": 10, "total": 8, "totalPage": 1 },
  "data": [
    {
      "_id": "e1",
      "title": "The End's Beginning",
      "episodeNumber": 1,
      "duration": 45,
      "videoUrl": "https://uremz-video-stream.b-cdn.net/media/videos/ep1.mp4",
      "thumbnail": "https://uremz-video-stream.b-cdn.net/thumbnails/ep1.jpg",
      "status": "PUBLISHED"
    }
  ]
}
```
